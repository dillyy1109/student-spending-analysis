# 📊 Student Spending Analysis

## Overview
This project analyzes student spending behavior to understand how students allocate their expenses and whether financial factors such as income influence spending habits.

## Objective
**Research Question:**  
How do student spending habits differ by income and lifestyle?

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Dataset
Kaggle Dataset:  
https://www.kaggle.com/datasets/sumanthnimmagadda/student-spending-dataset

## Key Insights
- Most students **overspend relative to their monthly income**
- Spending patterns are **consistent across gender and year in school**
- Payment method has **minimal impact on spending**
- Spending distribution is **approximately normal**

## Project Structure
- Data Overview
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Analysis & Visualization
- Conclusion

## Limitations
- Dataset may not include all income sources (e.g., savings, parental support)
- Results may not generalize to all student populations

## Conclusion
This project shows that overspending is a common pattern among students, highlighting the importance of financial awareness and planning.
